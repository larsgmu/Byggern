Detailed Description

This file contains function to enable CAN communication between Node1 and Node2. Node1 uses a MCP2515 chip as a stand-alone CAN controller. The communication between atmega162 and MCP2515 is done with SPI. Node1 uses a MCP2551 chip as CAN transceiver.
